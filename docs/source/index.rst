Welcome to AstroPipe's documentation!
======================================

**AstroPipe** is a Python library for surface brightness profile analysis of
astronomical images. It provides tools for image download, masking, background
estimation, morphology measurement, isophotal photometry, and visualization,
with a focus on low-surface-brightness science.
